# [기획안] Edu Sync AI: AI 기반 지능형 교육 오케스트레이션 솔루션

## 1. 프로젝트 개요
* **서비스명:** Edu Sync AI
* **슬로건:** "관리(Management)를 넘어 공명(Sync)으로, AI가 설계하는 교육의 미래"
* **핵심 가치:** 교육 현장의 행정적 번아웃 해소 및 데이터 기반의 초개인화 학습 경험 제공
* **주요 타겟:** 맞춤형 피드백이 필요한 교강사, 자기주도 학습 결손이 있는 수강생

---

## 2. 기획 배경 및 페인 포인트 (Pain Points)

### 2.1. 기존 LMS의 한계
* **단순 저장소:** 학습 자료의 업로드와 출석 확인 등 행정적 기능에 치중하여 실질적 교육 질 개선에 한계.
* **피드백 지연:** 교사 1인이 다수 학생의 주관식 답변이나 학습 상태를 실시간으로 파악하고 개별 피드백을 주기에 물리적 시간 부족.
* **단절된 학습 경험:** 학생 개개인의 이해도 차이를 반영하지 못하고 정해진 커리큘럼만 반복하는 일방향적 시스템.

### 2.2. Edu Sync AI의 해결책
* **AI 비서화:** 교사에게는 '채점 및 분석 비서'를, 학생에게는 '24/7 전담 튜터'를 제공하여 교육의 질 상향 평준화.
* **실시간 동기화(Sync):** 학습자의 인지 상태를 실시간으로 분석하여 즉각적인 학습 경로 수정 및 맞춤형 콘텐츠 제공.

---

## 3. 핵심 AI 기능 (Core AI Features)

| 기능명 | 대상 | 상세 내용 | AI 활용 기술 |
| :--- | :--- | :--- | :--- |
| **AI 피드백 엔진** | 교강사 | 서술형 과제 및 질문에 대한 1차 자동 채점 및 개인별 맞춤 피드백 초안 생성 | LLM (GPT-4o), Few-shot Prompting |
| **Adaptive Roadmap** | 수강생 | 퀴즈 결과 및 학습 이력을 분석하여 취약점 보완용 커스텀 로드맵 및 유사 문제 생성 | RAG (Retrieval-Augmented Generation) |
| **Smart Sync 알림** | 운영자 | 학습 이탈 징후나 성취도 급락이 보이는 수강생을 감지하여 관리자에게 '개입 적기' 알림 | 데이터 패턴 분석 및 정형 데이터 분류 |
| **수업 콘텐츠 생성기** | 교강사 | 수업 키워드 입력 시 맞춤형 퀴즈, 요약 노트, 강의 스크립트 자동 생성 | Text Generation & Summarization |

## 4. 서비스 아키텍처 (Technical Stack)

* **Frontend:** `React Native (Expo)` - iOS/Android 크로스 플랫폼 대응 및 빠른 배포 주기에 최적화.
* **Backend:** `Supabase` - PostgreSQL 기반 실시간 DB, Edge Functions를 통한 서버리스 AI 로직 처리.
* **Auth:** `Kakao Login API` - 국내 교육 현장의 사용자 편의성을 고려한 간편 인증 도입.
* **AI Engine:** `OpenAI API / LangChain` - 지능형 대화 처리 및 대규모 문서 분석 엔진.

## 5. 주요 유저 시나리오 (User Journey)

### [Scenario A: 교강사 - "업무의 80%를 AI에게"]
1. **자료 업로드:** 오늘 수업할 PDF/텍스트를 업로드하면 AI가 핵심 키워드와 예상 질문을 추출합니다.
2. **자동 피드백:** 학생들의 과제 제출 즉시 AI가 분석하여 성적 분포도와 개별 보충 학습 코멘트를 제안합니다.
3. **대시보드:** AI 분석을 통해 "현재 이 개념을 어려워하는 학생 그룹"을 파악하고 집중 코칭을 진행합니다.

### [Scenario B: 수강생 - "나만을 위한 AI 튜터"]
1. **맞춤형 학습:** 앱 접속 시 AI가 이전 학습 데이터를 기반으로 오늘의 최적화된 학습 목표를 제시합니다.
2. **실시간 질의응답:** 학습 중 궁금한 점을 질문하면 AI가 강의 맥락을 파악하여 실시간으로 답변해 줍니다.
3. **AI 리포트:** 단순 점수가 아닌, "나의 사고 과정 중 어느 부분이 개선되었는지" 정성적인 성장 지표를 확인합니다.

## 6. 기대 효과 및 차별점

* **실무 접합성:** 단순 플랫폼 구축이 아닌, 교사의 행정 업무를 실질적으로 경감하고 학생의 중도 포기를 막는 실효성 있는 도구.
* **기술적 완성도:** Supabase와 Expo를 결합한 고성능 아키텍처로 실시간성과 확장성 확보.
* **데이터 선순환:** 학습 데이터 축적에 따른 AI 피드백 정교화로 교육 서비스의 품질이 지속적으로 상향되는 구조.
