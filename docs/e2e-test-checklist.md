# EduSync AI — E2E 테스트 체크리스트

전체 플로우를 순서대로 확인합니다. **공모전·발표 시연은 웹(Vercel) 우선**을 권장합니다([데모 가이드](./demo-guide.md)). 개발 빌드(`npx expo run:ios` / `npx expo run:android`) 또는 로컬 웹(`npm run web`)에서도 동일 항목을 검증할 수 있습니다.

---

## 사전 조건

- [ ] `.env`에 `EXPO_PUBLIC_SUPABASE_URL`, `EXPO_PUBLIC_SUPABASE_ANON_KEY`, `EXPO_PUBLIC_KAKAO_NATIVE_APP_KEY`, `EXPO_PUBLIC_GEMINI_API_KEY` 설정 완료
- [ ] Supabase Dashboard → SQL Editor에서 `002_role_nullable.sql` 마이그레이션 실행 완료
- [ ] Supabase Edge Function `kakao-auth` 배포 완료
- [ ] Supabase Dashboard → Authentication → URL Configuration에 Redirect URI 등록

---

## 1. 인증 플로우

### 1-1. 신규 가입 (카카오 로그인)
- [ ] 로그인 화면에서 "카카오로 시작하기" 버튼 탭
- [ ] 카카오 인증 화면으로 이동
- [ ] 로그인 완료 후 **역할 선택 화면**으로 이동
- [ ] Supabase Dashboard → Authentication → Users에 신규 유저 생성 확인
- [ ] Supabase Dashboard → Table Editor → profiles에 `role = NULL` 레코드 확인

### 1-2. 역할 선택
- [ ] "교사" 카드 선택 → profiles.role = 'teacher' 업데이트 확인
- [ ] 교사 탭 화면(퀴즈 생성)으로 이동 확인
- [ ] 로그아웃 후 동일 계정 재로그인 → 역할 선택 화면 없이 바로 교사 탭 이동 확인

### 1-3. 학생 역할 확인
- [ ] 다른 카카오 계정으로 로그인 → "학생" 선택
- [ ] 학생 탭 화면(퀴즈 목록)으로 이동 확인

---

## 2. 교사 플로우

### 2-1. AI 퀴즈 생성
- [ ] 퀴즈 생성 탭에서 입장 코드 표시 확인
- [ ] 퀴즈 제목 입력
- [ ] 수업 자료 텍스트 50자 이상 입력
- [ ] "AI로 퀴즈 생성" 버튼 탭 → 로딩 스피너 + "AI가 퀴즈를 생성하고 있습니다..." 표시
- [ ] 생성 완료 후 퀴즈 목록 탭으로 자동 이동
- [ ] 퀴즈 목록에 새 퀴즈 카드 표시 (제목, 문제 수, 개념 태그)

### 2-2. 퀴즈 목록
- [ ] 생성된 퀴즈 카드 목록 표시
- [ ] 입장 코드 확인 (학생에게 공유할 코드)

### 2-3. 실시간 대시보드
- [ ] 대시보드 탭 진입 → "아직 응시한 학생이 없습니다" 메시지 표시
- [ ] 학생이 퀴즈 응시 후 → 대시보드 자동 갱신 확인 (새로고침 없이)
- [ ] 학생 카드 정답률 색상 확인 (≥80% 초록, 60~79% 파랑, <60% 빨강)
- [ ] 취약/강점 개념 태그 표시 확인

---

## 3. 학생 플로우

### 3-1. 수업 참여
- [ ] 수업 참여 탭에서 교사에게 받은 입장 코드 입력
- [ ] "참여하기" 버튼 탭 → "참여 완료!" 알림
- [ ] 퀴즈 탭에서 해당 수업의 퀴즈 목록 표시 확인

### 3-2. 퀴즈 응시
- [ ] 퀴즈 목록에서 퀴즈 선택 → 퀴즈 화면 이동
- [ ] 진행률 바 및 "N/M" 표시 확인
- [ ] 선택지 탭 → 강조 표시 확인
- [ ] "제출" 버튼 탭

**정답 시:**
- [ ] 초록 "정답입니다!" 피드백 표시
- [ ] "다음 문제" 버튼으로 이동

**오답 시:**
- [ ] 빨간 "오답입니다." 피드백 + 정답 선택지 초록 강조
- [ ] "힌트 보기" 버튼 탭 → AI 힌트 로딩 → 노란 힌트 카드 표시
- [ ] "다음 문제" 버튼으로 이동

### 3-3. 결과 화면
- [ ] 마지막 문제 완료 후 결과 화면 (정답률 %, 정답 수/전체) 표시
- [ ] "목록으로 돌아가기" 버튼 동작 확인

### 3-4. 학습 리포트
- [ ] 리포트 탭 → 전체 정답률, 응시 퀴즈 수 요약 카드 표시
- [ ] 개념별 정답률 프로그레스 바 표시 (취약 개념 빨간색 강조)
- [ ] 응시 이력 목록 (퀴즈명, 점수 배지) 표시
