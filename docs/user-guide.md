## 1. 시작하기 (공통)

### 1.1 로그인

1. 앱 접속 후 **카카오 로그인** 버튼 클릭
2. 카카오 계정으로 인증 완료
3. **최초 로그인 시** 역할 선택 화면이 표시됩니다

### 1.2 역할 선택

| 역할 | 설명 |
|------|------|
| **교사** | AI 퀴즈 생성 및 학생 학습 관리 |
| **학생** | 수업 참여 및 AI 퀴즈 응시 |

> 역할은 프로필 화면에서 언제든지 변경할 수 있습니다.

### 1.3 로그아웃 및 역할 변경

- **프로필** 탭 → **역할 변경** 버튼: 역할 선택 화면으로 이동
- **프로필** 탭 → **로그아웃** 버튼: 로그인 화면으로 이동

## 2. 교사 가이드

### 2.1 화면 구성

| 탭 | 기능 |
|----|----|
| 퀴즈 생성 | AI로 퀴즈 생성 및 수업 관리 |
| 퀴즈 보관함 | 생성된 퀴즈 목록 조회 |
| 대시보드 | 실시간 학생 응답 현황 |
| 프로필 | 내 정보 및 설정 |

---

### 2.2 수업(클래스룸) 만들기

1. **퀴즈 생성** 탭 접속
2. 화면 상단에 표시된 **입장 코드** 확인
   - 입장 코드는 수업마다 자동 생성됩니다
   - 학생들에게 이 코드를 공유하세요

---

### 2.3 AI 퀴즈 생성

1. **퀴즈 생성** 탭에서 퀴즈 제목 입력
2. 학습 내용(지문, 개념 등) 텍스트 입력
3. **AI 퀴즈 생성** 버튼 클릭
4. AI가 자동으로 객관식 문제를 생성합니다
5. 생성된 퀴즈 확인 후 **수업에 출제** 버튼으로 배포

> AI 생성에는 수초가 소요됩니다. 생성 중 화면을 벗어나지 마세요.

---

### 2.4 퀴즈 보관함

- **퀴즈 보관함** 탭에서 이전에 생성한 퀴즈 목록 확인
- 퀴즈 카드의 **출제** 버튼으로 재배포 가능
- 퀴즈 카드의 **삭제** 버튼으로 퀴즈 삭제

---

### 2.5 실시간 대시보드

1. **대시보드** 탭 접속
2. 진행 중인 퀴즈에 참여한 학생들의 응답 현황 실시간 확인
   - 응시 학생 수
   - 문항별 정답률
   - 개념별 취약점 현황
3. 데이터는 학생이 제출할 때마다 자동 갱신됩니다

## 3. 학생 가이드

### 3.1 화면 구성

| 탭 | 기능 |
|----|-----|
| 퀴즈 | 수업 참여 및 퀴즈 응시 |
| 리포트 | 나의 학습 현황 분석 |
| 프로필 | 내 정보 및 설정 |

---

### 3.2 수업 참여하기

1. **퀴즈** 탭 접속
2. **수업 참여** 버튼 클릭
3. 교사에게 받은 **입장 코드** 입력
4. 참여 완료 후 해당 수업의 퀴즈가 목록에 표시됩니다

> 한 번 참여한 수업은 이후 자동으로 목록에 유지됩니다.

---

### 3.3 퀴즈 응시

1. **퀴즈** 탭에서 응시할 퀴즈 선택
2. 문제를 읽고 보기 중 정답 선택
3. 선택 즉시 정답 여부와 **AI 피드백** 확인
4. 모든 문제 완료 후 결과 화면에서 최종 점수 확인

> AI 피드백은 오답 문항에 대해 개념 설명과 함께 제공됩니다.

---

### 3.4 학습 리포트

**리포트** 탭에서 나의 학습 데이터를 확인할 수 있습니다.

| 항목 | 설명 |
|------|------|
| 전체 정답률 | 응시한 모든 퀴즈의 누적 정답률 |
| 응시 퀴즈 수 | 지금까지 응시한 퀴즈 총 개수 |
| 개념별 정답률 | 개념 태그별 정확도 (60% 미만 시 취약 표시) |
| 응시 이력 | 퀴즈별 점수 및 정답 현황 |

> 개념별 정답률 그래프에서 **빨간색**으로 표시된 항목이 취약 개념입니다. 해당 개념을 집중적으로 복습하세요.

## 4. 자주 묻는 질문 (FAQ)

**Q. 입장 코드를 잘못 입력했어요.**
> 코드는 대소문자를 구분하지 않습니다. 교사에게 코드를 다시 확인 요청하세요.

**Q. 퀴즈 목록에 아무것도 안 떠요.**
> 교사가 퀴즈를 아직 출제하지 않았거나 수업에 참여하지 않은 상태입니다. 수업 참여 후 새로고침 해보세요.

**Q. 역할을 잘못 선택했어요.**
> 프로필 탭 → **역할 변경** 버튼을 눌러 다시 선택할 수 있습니다.

**Q. AI 퀴즈 생성이 너무 오래 걸려요.**
> AI 생성은 보통 5~15초 소요됩니다. 네트워크 상태를 확인하고 기다려 주세요.

**Q. 로그인이 안 돼요.**
> 카카오 계정이 정상적인지 확인하고, 브라우저 캐시를 삭제 후 재시도 해보세요.
