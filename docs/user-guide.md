## 1. 시작하기 (공통)

### 1.1 로그인

1. 앱(또는 웹) 접속 후 **카카오 로그인** 버튼을 누릅니다.
2. 카카오 계정으로 인증을 완료합니다.
3. **최초 로그인 시** 역할 선택 화면이 표시됩니다.

### 1.2 역할 선택

| 역할 | 설명 |
|------|------|
| **교사** | AI 퀴즈 생성 및 학생 학습 관리 |
| **학생** | 입장 코드로 수업 참여 후 AI 퀴즈 응시·리포트 확인 |

> 역할은 프로필 화면에서 언제든지 변경할 수 있습니다.

### 1.3 로그아웃 및 역할 변경

- **프로필** 탭 → **역할 변경** 버튼: 역할 선택 화면으로 이동합니다.
- **프로필** 탭 → **로그아웃** 버튼: 로그인 화면으로 이동합니다.

## 2. 교사 가이드

### 2.1 화면 구성

| 탭 | 기능 |
|----|------|
| 퀴즈 생성 | AI로 퀴즈를 만들고, 입장 코드를 확인합니다 |
| 퀴즈 목록 | 이미 만든 퀴즈 제목·문항 수·개념 태그를 봅니다 |
| 대시보드 | 실시간 학생 응답 현황 |
| 프로필 | 내 정보 및 설정 |

---

### 2.2 수업(클래스룸)과 입장 코드

1. **퀴즈 생성** 탭으로 이동합니다.
2. 화면에 표시된 **입장 코드**를 확인합니다.
   - 교사 계정에 연결된 기본 수업이 있으며, **입장 코드는 그 수업마다 고정**됩니다.
   - 학생에게 이 코드를 알려 주면 해당 수업에 참여할 수 있습니다.

---

### 2.3 AI 퀴즈 생성

1. **퀴즈 생성** 탭에서 퀴즈 제목을 입력합니다.
2. 학습 내용(지문, 개념 등)을 **50자 이상** 텍스트로 입력합니다.
3. **AI 퀴즈 생성** 버튼을 누릅니다.
4. AI가 객관식 문제를 생성합니다. 완료 후 알림에서 확인하면 **퀴즈 목록**으로 이동할 수 있습니다.

> 생성된 퀴즈는 **저장과 동시에 해당 수업에 연결**됩니다. 별도의 「출제」 단계는 없습니다. 해당 수업에 이미 참여한 학생은 **퀴즈** 탭에서 바로 퀴즈를 볼 수 있습니다.

> AI 생성에는 수초가 걸릴 수 있습니다. 생성 중에는 화면을 벗어나지 않는 것이 좋습니다.

---

### 2.4 퀴즈 목록

- **퀴즈 목록** 탭에서 만든 퀴즈를 카드 형태로 확인합니다.
- 상단에 **입장 코드**가 다시 표시되므로 학생에게 공유할 때 활용할 수 있습니다.

---

### 2.5 실시간 대시보드

1. **대시보드** 탭으로 이동합니다.
2. 수업에 참여해 응시한 학생의 현황을 실시간으로 확인합니다.
   - 응시 완료 인원
   - 평균 정답률
   - 학생별 정답률 카드
   - 개념 태그 기준 취약·강점 요약
3. 학생이 답을 제출할 때마다 데이터가 갱신됩니다(Supabase Realtime).

## 3. 학생 가이드

### 3.1 화면 구성

| 탭 | 기능 |
|----|------|
| 퀴즈 | 입장 코드로 수업 참여 + 참여한 수업의 퀴즈 목록·응시 |
| 리포트 | 나의 학습 현황 분석 |
| 프로필 | 내 정보 및 설정 |

---

### 3.2 수업 참여하기

1. **퀴즈** 탭으로 이동합니다.
2. 상단 **입장 코드** 칸에 교사에게 받은 코드를 입력합니다.
3. **참여하기**를 누릅니다. 성공하면 같은 화면에 초록색 안내가 잠시 표시되고, **내 퀴즈** 영역에 퀴즈가 나타납니다. 아직 수업에 참여하지 않았다면 **내 퀴즈** 안에 입장 안내만 보입니다.
4. 코드가 틀리면 빨간색 오류 문구로 바로 알려 줍니다.

> 한 번 참여한 수업은 이후에도 목록에 유지됩니다. 다른 수업에 들어가려면 같은 화면에서 코드를 다시 입력하면 됩니다.

---

### 3.3 퀴즈 응시

1. **퀴즈** 탭의 목록에서 응시할 퀴즈를 선택합니다.
2. 문제를 읽고 보기 중 하나를 고른 뒤 **제출**합니다.
3. **정답**인 경우 안내 메시지 후 다음 문제로 넘어갑니다.
4. **오답**인 경우 **다음 문제**로 바로 넘어갈 수 있고, 원하면 **힌트 보기**로 AI 소크라테스식 힌트를 받을 수 있습니다. 힌트를 본 뒤에도 **다음 문제**로 진행합니다.
5. 모든 문제를 끝내면 최종 점수 화면이 표시됩니다.
6. 한 번 끝까지 응시한 퀴즈는 목록에서 **취소선·응시 완료**로 표시되며, 다시 풀 수 없습니다(통계·리포트 신뢰도 유지).

> 오답 시 화면에서 정답 보기가 강조될 수 있습니다. AI 힌트는 정답을 직접 알려주지 않고 생각할 수 있는 단서를 제공하는 것을 목표로 합니다.

---

### 3.4 학습 리포트

**리포트** 탭에서 나의 학습 데이터를 확인할 수 있습니다.

| 항목 | 설명 |
|------|------|
| 전체 정답률 | 응시한 모든 문항 기준 누적 정답률 |
| 응시 퀴즈 수 | 응시 이력이 있는 퀴즈 수 |
| 개념별 정답률 | 개념 태그별 정확도 (60% 미만 시 취약 표시) |
| 응시 이력 | 퀴즈별 점수 및 정답 현황 |

> 개념별 정답률에서 **빨간색**으로 표시된 항목은 취약 개념으로 보면 됩니다.

## 4. 자주 묻는 질문 (FAQ)

**Q. 입장 코드를 잘못 입력했어요.**

> 코드는 대소문자를 구분하지 않습니다. 교사에게 코드를 다시 확인해 달라고 하세요.

**Q. 퀴즈 목록에 아무것도 안 떠요.**

> 아직 **입장 코드로 수업에 참여하지 않았거나**, 교사가 퀴즈를 만들지 않은 경우일 수 있습니다. **퀴즈** 탭 상단에서 코드로 참여한 뒤 목록을 확인해 보세요.

**Q. 같은 퀴즈를 다시 풀고 싶어요.**

> 모든 문항을 제출 완료한 퀴즈는 **통계·리포트 신뢰도**를 위해 다시 풀 수 없습니다. 목록에서 **취소선·응시 완료**로 표시됩니다.

**Q. 역할을 잘못 선택했어요.**

> 프로필 탭 → **역할 변경**으로 다시 선택할 수 있습니다.

**Q. AI 퀴즈 생성이 너무 오래 걸려요.**

> 보통 수 초~15초 정도 걸릴 수 있습니다. 네트워크 상태를 확인하고 잠시 기다려 주세요.

**Q. 로그인이 안 돼요.**

> 카카오 앱·계정 상태와, 카카오 개발자 콘솔의 Redirect URI 설정을 확인해 보세요. 웹에서는 브라우저 캐시 삭제 후 재시도할 수 있습니다.
